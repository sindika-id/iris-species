# Code of Conduct

## Our Commitment

We are committed to providing a respectful, inclusive, and professional environment for everyone participating in this project.

This repository is intended for educational, research, and collaborative software development purposes. All contributors are expected to interact in good faith and with mutual respect.

## Expected Behavior

Participants are expected to:

- Use respectful and constructive language
- Be considerate of different perspectives and experience levels
- Accept feedback gracefully
- Focus discussions on technical and project-related matters
- Follow professional standards of communication

## Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment, discrimination, or abusive language
- Personal attacks or insults
- Trolling, inflammatory comments, or deliberate disruption
- Publishing private or confidential information without consent

## Enforcement

Project maintainers reserve the right to review and moderate all contributions, issues, and discussions.  
Any behavior that violates this Code of Conduct may result in removal of content or restriction from participation.

## Reporting Issues

If you experience or witness unacceptable behavior, please report it privately to the repository maintainers.

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, discussions, and other forms of communication related to this repository.

---

This Code of Conduct is adapted from common open-source best practices and applies to this repository only.
